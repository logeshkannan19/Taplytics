# Taplytics Roadmap ⌨️

Professionalizing the typing experience for developers and enthusiasts.

## 🟢 Phase 1: Core Experience (Complete)
- [x] Precise WPM & Accuracy calculation.
- [x] Real-time performance graph (Canvas based).
- [x] Auditory feedback system.
- [x] Dark mode first design.

## 🟡 Phase 2: Personalization & Training
- [ ] **Custom Word Banks**: Upload your own text or use technical snippets (React, Rust, etc.).
- [ ] **Adaptive Difficulty**: Algorithmically focus on words or characters where the user is slowing down.
- [ ] **Heatmaps**: Visualizing key-by-key accuracy and delay.

## 🔴 Phase 3: Global & Social
- [ ] **Leaderboards**: Competitive modes with verified results.
- [ ] **Ghost Mode**: Type against your previous personal bests in real-time.
- [ ] **Multiplayer**: Live races against other users.
